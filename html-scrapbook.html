Images with img element

Images brighten up your page and make it more personable. A page without images can be extremely dull!
On HTML, you can embed an image onto the page with the img tag. Technically speaking, images do not actually go on the page; they are linked to the page.

There are two required attributes to embed an image:
  1. SRC. This specifies path to the image.
  2. alt. This specifies an alternate text for the image, if image cannot be displayed.

Tables

The table tag defines an HTML table. In HTML, tables can be simple or complex

An HTML table consists of one table element and one or more of the tr, th and td elements.

The tr element defines a table row, the th element defines a table header and the td element defines a table cell. An HTML table can also include other elements.
